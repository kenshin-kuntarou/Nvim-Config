# Nvim-Config

To Install:

```
git clone https://github.com/kenshin-kuntarou/Nvim-Config ~/.config/nvim/
```

Plugins:

* Black
* Css-lsp
* Goimports
* Golines
* Gopls
* Html-lsp
* Lua-language-server
* Mypy
* Pyright
* Ruff
* Rufo
* Rubocop
* Stylua 
* Solargraph

Install Plugins:

```
MasonInstallAll
```

Or:

```
MasonInstall black css-lsp goimport golines gopls html-lsp lua-language-server mypy pyright ruff rufo rubocop stylua solargraph 
```
